AirBnB Clone

Una replica di AirBnB

Il progetto prevedeva di prendere spunto dal sito di AirBnB per averne una copia funzionante, sia lato front-end che back-end.
Lato front-end è stato usato Sass per la gestione dello stile e jQuery come libreria Javascript.
Per il back-end è stato usato il framework Laravel per tutta la logica lato server. Lato database è stato utilizzato MySQL.
Come servizi terzi è stato utilizzato Algolia Places per inserire le mappe con i pin di ciascuna casa e Braintree per simulare la gestione dei pagamenti tramite carta di credito.

Linguaggi utilizzati: HTML5 / CSS3 / JAVASCRIPT / PHP.
